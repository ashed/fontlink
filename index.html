FontLink

Usage • Downloads • Development

FontLink is a program for Unix-like systems that allows you to use fonts without having to install them.

Usage

The FontLink's interface consists of two lists. The left one contains font sets; the right one shows fonts of the selected set.

To add fonts to a set, use the button below the font list, the context menu, or drop them from your file manager anywhere on the FontLink's window. You can add the same font in several sets. Disabling or removing such a font from one set will not unlink it if it's still enabled in others.

If a font that is already installed somewhere on your system, it will appear as “inactive” and its checkbox will always be enabled. Check its pop-up tooltip to see where it is installed.

It's recommended to set up your fonts before running a program that will use them, because not all programs can update the list of fonts at runtime.

If you want to add FontLink to autostart, don't forget to add --minimized or -m argument to start the program with the hidden window.

Downloads

The most recent version of FontLink is 1.0.1.

  • fontlink_1.0.1-1_all.deb — Debian/Ubuntu package

  • fontlink-1.0.1.tar.gz — source code

You can use alien to install the DEB package on non-DEB systems.

Another way is to install from the sources using make. Read INSTALL.md for detailed instructions.

It's possible to use FontLink without installation: unpack the source code tarball anywhere and execute bin/fontlink. Still, you should ensure you have all the dependencies installed.
